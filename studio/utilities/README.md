# Сторонние утилиты

В TRIK Studio доступны три сторонние утилиты:

1. [PuTTY](putty.md).
2. [WinSCP](winscp.md).
3. [TRIK Gamepad](../../gamepad/about/utility.md).

Для запуска утилиты зайдите в меню `Инструменты → Сторонние утилиты`.
