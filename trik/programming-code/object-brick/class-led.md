# Класс «led»

Предоставляет управление светодиодом на корпусе робота.

| Метод                         | Описание                                |
| ----------------------------- | --------------------------------------- |
| [red](class-led.md#red)       | Включает светодиод в режим «красный».   |
| [green](class-led.md#green)   | Включает светодиод в режим «зеленый».   |
| [orange](class-led.md#orange) | Включает светодиод в режим «оранжевый». |
| [off](class-led.md#off)       | Выключает светодиод.                    |

## red

Включает светодиод в режим «красный».

#### Синтаксис

{% tabs %}
{% tab title="Python" %}
```

```
{% endtab %}

