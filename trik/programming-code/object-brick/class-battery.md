# Класс «battery»

Предоставляет доступ к информации о батарее или блоке питания.

## readVoltage

Возвращает текущий вольтаж батареи (или блока питания) в вольтах.

#### Синтаксис

{% tabs %}
{% tab title="Python" %}
```
brick.battery().readVoltage()
```
{% endtab %}

{% tab title="JavaScript" %}
```javascript
brick.battery().readVoltage();
```
{% endtab %}
{% endtabs %}
